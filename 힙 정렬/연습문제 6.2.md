6-2-2 MAX-HEAPIFY 프로시저를 참고해 최소 힙을 위한 프로시저 MIN-HEAPIFY(A,i)의 의사코드를 작성하라

-
MIN-HEAPIFY(A, i)
  l = LEFT(i)
  r = RIGHT(i)

  if l<=A.heap-size and A[l] < A[i]
     smallest = l
  else smallest = i

  if r<=A.heap-size and A[r] < A[smallest]
     smallest = r

  if smallest != i
      exchange A[i] with A[smallest]
      MIN-HEAPIFY(A, smallest)

- 수행시간은 O(logn)이다.
